Avances del Servidor de Minecraft

Tras el intento fallido de tener nuestro servidor nos encontramos trabajando de nuevo en el con la actualización más reciente, Aunque esperaremos a la versión 1.17, el server se mantendrá activo 24/7 con ayuda de nuestro host.
Como anteriormente probamos, logramos hacer una integración con la versión de Java (PC) y Bedrock (Celulares) dando la posibilidad de que nadie se quede afuera.
Ademas estoy preparando la siguiente actualización (2.1) de El Roles (Bot de nuestra comunidad de discord) para hacer un sistema de economía integrado con el servidor de minecraft en donde puedan obtener ítems coleccionables y rangos.
También brindamos una membresía para tener distintas funciones en el servidor, las cuales se podrán abonar por Paypal, Mercado pago, CVU y Patreon (Próximamente).

Actualizacion de El Roles 2.0

Tras meses de trabajo El Roles fue actualizado a su versión 2.0, en sus novedades está la corrección de bugs de algunos comandos, sistema de bienvenida con imágenes, sistema de niveles con base en MySQL, Sistema de economía con algunas correcciones, funciones ocultas, funciones de administración, comandos secretos, sistema de doble defensa activado, sistema de verificación, traducción a nuevos idiomas y sistema de tickets.

Boosteos del servidor

Gracias a las mejoras de 🎆| Server Booster. El servidor ahora tiene esta tarjeta de ppresentaciónal invitar personas a nuestra comunidad. Ademas de el nuevo icono animado, canales de mmúsicaa 128kbps y emojis animados(PPróximamentelos estoy subiendo) para aquellos que sean usuarios Nitro.
